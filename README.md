#  FE Config package

## Introduction

> 프론트엔드 프로젝트 빌딩 시 자주 사용되는 환경설정 패키지 모음입니다

## Used Specific Library

> `typescript`, `jest`, `eslint`, `prettier`

## Initial Setting

- `.npmrc` : 패키지 배포를 위한 npm 환경변수

## Getting Started

```bash
# yarn 기반
yarn publish # 첫 배포
yarn publish --patch # 배포 시 버전업
```
